# team04

## Frontend

1. Make sure you're in the directory `frontend`:

```
cd frontend
```

2. Install dependencies:

```
npm ci
```

3. Fire up the React development server:

```
npm start
```

Before you install a new package, always make sure you are in the directory `frontend`.
